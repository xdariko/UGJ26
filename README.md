# Ritebound

Ritebound — игра, созданная для **UralGameJam 2026**.

После исчезновения ведьмы её фамильяр остаётся рядом с её мрачным наследием.  
Чтобы раскрыть, что именно она оставила после себя, игроку предстоит собирать эфир, добываемый из демонических существ, и проводить ритуалы.
